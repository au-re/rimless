# Rimless

Rimless makes event-based communication easy with a promise-based API wrapping [postMessage](https://developer.mozilla.org/en-US/docs/Web/API/Window/postMessage).

Rimless works with **iframes**, **web workers**, and **Node.js worker threads**.

You can use `rimless` to call remote procedures, exchange data, or expose local functions across those contexts.
The browser-based examples below cover iframe and web worker integrations.
