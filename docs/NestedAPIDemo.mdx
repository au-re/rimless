# Nested Methods

Rimless supports nested method schemas. Remote APIs can expose deeply nested
functions and you can access them with property chaining on `connection.remote`.

For example, a worker may expose:

```ts
{
  user: {
    profile: {
      get: () => ({ name: "Alice", age: 30 })
    }
  }
}
```

You can call this from the host with `connection.remote.user.profile.get()`.
